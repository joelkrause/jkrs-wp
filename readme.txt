== Frequently Asked Questions ==

= Compiling =
* Uses CodeKit
* CSS file should compile to /lib/styles/css/main.min.css
* JS file uses CodeKit prepends (to help save server requests) - add your external JS files inside the src folder and add the file to the append at the top of main.js
* Your js file should compile to /lib/scripts/scripts.min.js

== Resources ==
* TweenMax (https://greensock.com/tweenmax)
* ScrollMagic (scrollmagic.io)
* animation.gsap.js file (used for TweenMax/ScrollMagic animations)
* slick
